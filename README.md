# 👁️ Visão Computacional com OpenCV e EasyOCR

Projeto desenvolvido para estudos de **Visão Computacional** utilizando **Python**, **OpenCV** e **EasyOCR**. O objetivo é demonstrar conceitos básicos de processamento de imagens e reconhecimento óptico de caracteres (OCR), aplicados à leitura de placas de veículos.

## 🚀 Funcionalidades

* Leitura e redimensionamento de imagens.
* Conversão para escala de cinza.
* Marcação de Região de Interesse (ROI).
* Reconhecimento de caracteres com EasyOCR.
* Exibição da placa detectada e do nível de confiança.

## 🛠️ Tecnologias

* Python
* OpenCV
* EasyOCR
* Google Colab

## 📦 Instalação

```bash
pip install opencv-python easyocr
```

## ▶️ Como Executar

1. Clone o repositório.
2. Instale as dependências.
3. Adicione as imagens `cenario.jpg` e `carro.jpg`.
4. Execute o notebook `Opencv.ipynb` no Google Colab ou Jupyter Notebook.

## 📚 Conceitos Aplicados

* Processamento de imagens
